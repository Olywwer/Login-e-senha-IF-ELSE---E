#  Sistema de Login Simples

Este projeto consiste em uma aplicação web simples desenvolvida com **HTML, CSS, JavaScript e Bootstrap**, que simula um sistema de autenticação de usuário.


##  Funcionalidades

* Solicita ao usuário:

  * Nome de usuário
  * Senha

* Realiza validações com base nas seguintes regras:

| Situação                          | Resultado                |
| --------------------------------- | ------------------------ |
| Usuário e senha corretos          | ✅ Login bem-sucedido     |
| Usuário correto e senha incorreta | ⚠️ Senha incorreta       |
| Usuário inexistente               | ❌ Usuário não encontrado |
